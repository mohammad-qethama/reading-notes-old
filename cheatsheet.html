Linux/Unix Terminal commands cheat-sheet

pwd (print working directory)
- used to show the absoulete path for current directory
ls (list)
- used to show the content of current directory
ls -l
- used to show detailed content of current directory
ls [path]
- used to show detailed content of directory in the path
cd (change directory)
-used to move in directories and change location

NOTES

  • linux is case sensetive
  • linux can have files without extention
  • everything is a file under linux
  • -a is an option used to show hidden file when it used with the command ls
